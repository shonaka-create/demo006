SERVICE 01

ALC新築施工

新築建物へのALCパネル（軽量気泡コンクリートパネル）取付工事を専門に行っています。構造図・施工図の確認から始まり、墨出し・パネル加工・取付・コーキングまで一貫して対応します。

登録基幹技能士の代表が全工程を管理するため、高精度かつ安全な施工を実現。ゼネコン様・建設会社様の工程に合わせた柔軟なスケジュール対応も可能です。

✓ 構造図・施工図の確認と施工計画の立案
✓ 墨出し・ハット鋼取付・ALCパネル加工・取付
✓ コーキング（シーリング）処理・防水
✓ 工程管理・安全管理・品質検査
✓ マンション・オフィスビル・工場・倉庫など幅広い建物に対応
SERVICE 02

ALC改修工事

経年劣化したALCパネルの補修・打ち替え・防水処理を行います。ひび割れ・欠損・シーリングの劣化など、様々な劣化症状に対応し、建物の安全性と美観を長期にわたって維持します。

既存建物の状況を丁寧に調査・診断し、最適な改修方法をご提案。お客様の予算と工期に合わせた計画的な改修工事をご提供します。

✓ 外壁ALCパネルの現状調査・劣化診断
✓ パネルのひび割れ補修・欠損補修
✓ ALCパネルの打ち替え（新品交換）
✓ シーリング打ち替え・増し打ち
✓ 防水処理・撥水コーティング
SERVICE 03

元請けサポート

ゼネコン・中堅建設会社様の協力業者として、工程管理から人材手配まで幅広くサポートします。ALC工事の専門知識を持つチームが、現場の司令塔として機能します。

2班体制による柔軟な人員配置で、急な工程変更や増員要請にも対応可能。現場代理人の手を煩わせることなく、ALC工事を安心してお任せいただけます。

✓ 施工計画の立案・工程調整
✓ 施工管理（品質・工程・安全）
✓ 材料発注・数量算出サポート
✓ 関係機関への書類作成補助
✓ 緊急時・追加工事への迅速対応

ABOUT ALC

ALCとは？

私たちが専門とするALC（軽量気泡コンクリート）についてご説明します

ALC（Autoclaved Lightweight aerated Concrete）とは、珪石・セメント・石灰・アルミ粉末などを原料として製造される軽量気泡コンクリートのことです。

旭化成「ヘーベル」をはじめとするALCパネルは、マンション・ビル・工場・倉庫など様々な建物の外壁・間仕切り壁として広く使用されています。

D-ONEは、このALC工事に特化した専門会社として、高い施工精度と豊富な実績を持っています。

🔥
耐火性
国土交通大臣認定の耐火・準耐火構造
🔇
遮音性
気泡構造による優れた遮音・防音性能
🌡️
断熱性
省エネ効果の高い断熱・保温性能
⚖️
軽量性
コンクリートの約1/4の軽さで施工性◎
